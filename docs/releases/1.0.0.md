# Codex Pulse 1.0.0

**See how Codex thinks.**

Codex Pulse is a native macOS menu-bar monitor for live Codex reasoning telemetry. Version 1.0 turns every visible Thread into a stable, color-coded reasoning curve and every Turn into an inspectable workload node.

## Highlights

- One compact multi-Thread logarithmic reasoning timeline.
- Turn node area represents total token volume.
- Tiny local anomaly/status accents preserve Thread identity.
- Click-through reasoning-call distribution and Token Mix inspector.
- Realtime app-server reconciliation with lazy rollout detail caching.
- `1h` to `24h` macOS-style history wheel.
- Resource-aware lifecycle with one app-server, incremental JSONL parsing, and full Charts teardown while closed.
- Universal build for Apple Silicon and Intel Macs.

## Install

```bash
HOMEBREW_CASK_OPTS="--no-quarantine" brew install --cask soundadam/tap/codex-pulse
open -a "Codex Pulse"
```

The release is hardened-runtime and ad-hoc signed, but not yet Apple-notarized. The attached `.sha256` file verifies the release archive. See the [release documentation](../release.md) for details.

# Codex Pulse 1.0.1

Codex Pulse 1.0.1 makes the menu-bar presence immediately legible and establishes the project's open-source terms.

## Highlights

- A native pulse icon replaces the `Cdx` abbreviation.
- Plain-language states show `Idle`, the number of running Turns, detected issues, or a synchronization problem.
- Hover and accessibility text always identify Codex Pulse and its current state.
- The source is licensed under `AGPL-3.0-only`.

## Install

```bash
brew install --cask soundadam/tap/codex-pulse
xattr -dr com.apple.quarantine "/Applications/Codex Pulse.app"
open -a "Codex Pulse"
```

The release remains hardened-runtime and ad-hoc signed, but not yet Apple-notarized.
